# api-resp

适用于 thinkphp >= 6.0.0 的统一响应状态码类库

## 主要特性

* 轻量级
* 错误规范
* 适配性强
* 灵活响应
* 支持链写法

## 安装

~~~php
composer require mhhex/api-resp
~~~
安装完成后，会在项目的 config 文件夹里面自动生成 apiresp.php 配置文件。这个配置文件的作用是为不同的状态码提供对应的默认消息描述。

### 声明

~~~php
use mhhex\ApiResp;
~~~

### 示例

~~~php
// 示例一
$json = (new ApiResp())
    ->build();

// 示例二
$json = (new ApiResp())
    ->setStatusCode(0)
    ->build();

// 示例三
$json = (new ApiResp())
    ->setStatusCode(0)
    ->setMessage('成功')
    ->setTotal(100)
    ->setPageNum(1)
    ->setPageSize(10)
    ->setItem([
        ['id' => 1, 'name' => 'Item 1'],
        ['id' => 2, 'name' => 'Item 2'],
        // ... 更多数据项
    ])
    ->build();

// 如果没有通过 ->setMessage 方法显式地设置消息描述，
// 类库会自动根据设置的状态码去 apiresp.php 配置文件中查找对应的消息描述。
// 若配置文件中没有针对该状态码的配置，则会显示默认的 未知状态码 作为消息描述。
~~~
